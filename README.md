## How to build

1. Clone the repository

		$ git clone https://github.com/mightyidler/CKLibrary.git
		
2. Install pods

	    $ cd CKLibrary
	    $ pod install

3. Open the workspace in Xcode

	    $ open "CKLibrary.xcworkspace"

4. Compile and run the app in your simulator

## Requirements

- Xcode 11
- iOS 13+
- Swift 5
